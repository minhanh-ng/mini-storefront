# Mini Storefront coding project 4 

## Setup instructions
In terminal: 
1. cd Documents
2. npx create-next-app@latest mini-storefront (no TypeScript, Yes Eslint, Yes TailWind CSS, use src/, Yes App Router, No customize alias)
3. cd mini-storefront
4. npm install
5. npm run dev -> choose http://localhost:3000
